\##  Descripción del Proyecto

Este proyecto consiste en el desarrollo de la interfaz gráfica del compilador MicroC, construida con C# y Windows Forms. Implementa un entorno visual en modo oscuro utilizando la librería ReaLTaiizor, orientado a la lectura, edición y almacenamiento de archivos de código fuente (`\*.c`). La estructura base, compuesta por un editor de código y una consola de mensajes, servirá como fundamento para la integración futura de los módulos de análisis léxico, sintáctico y semántico.



\##  Tecnologías Utilizadas

\* \*\*Lenguaje:\*\* C# (.NET 10.0)

\* \*\*Framework Visual:\*\* Windows Forms

\* \*\*Librería de UI:\*\* ReaLTaiizor (Tema Crown)

\* \*\*Control de Versiones:\*\* Git / GitHub



\##  Instrucciones de Ejecución



\### Opción 1: Ejecutable (Recomendado)

1\. Ve a la sección de \*\*Releases\*\* en este repositorio.

2\. Descarga el archivo `.zip` adjunto en la versión \*\*v1.0-precompilador\*\*.

3\. Extrae el contenido en una carpeta de tu computadora.

4\. Haz doble clic en el archivo `.exe` para iniciar la aplicación (no requiere instalación).



\### Opción 2: Desde el Código Fuente

1\. Clonar este repositorio en tu máquina local.

2\. Abrir la solución (`.sln`) ubicada en la carpeta `/src/` utilizando \*\*Visual Studio 2026\*\*.

3\. Asegurarse de tener instalada la carga de trabajo de "Desarrollo de escritorio de .NET".

4\. Presionar `F5` para compilar y ejecutar la aplicación.
